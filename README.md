
    缩写对应：

    - 有颜色的物体，仅用作交换颜色：cT
    - 有颜色的物体，提供颜色而不改变自身：cTsp
    - 有颜色的门，变换为特定颜色时打开：cD
    - NPC：npc

    有颜色的门，颜色表示为能使其打开的那一种。
